# Inventory and Sales System [OFFLINE]

Inventory and Sales System made with Spring Boot and JavaFX.
This application uses an embedded H2 database, making it suitable for desktop use and offline operation without requiring any external database server.

## Requirements
- Java 17

## Build
To compile and build the project

```bash
mvn clean install
```
## Run Jar
After build, the generated file is in `target` folder
```bash
java -jar target/rm-hometileles.jar
```
